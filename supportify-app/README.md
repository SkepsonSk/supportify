## Nest Installation

```bash
$ npm install -g @nestjs/cli
$ npm install -g webpack webpack-cli
$ npm link webpack
$ npm link webpack-cli
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
